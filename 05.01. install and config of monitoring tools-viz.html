Monitoring Foundations

This section outlines the fundamental principles of network monitoring. Understanding the "four jobs" of monitoring ensures a balanced defense posture, while the syslog severity breakdown dictates how we classify and react to events. A healthy monitoring diet requires elements from all these foundational areas.

📦

Packets

What bytes crossed the wire? Provides raw, unsummarized truth of network traffic.

📊

Metrics

How busy or healthy is the device? Tracks performance over time (CPU, RAM, bandwidth).

📝

Logs

What did the device just tell us? Application and system events mapped to time.

🚨

Alerts

Which of the above demands a human? Thresholds and signatures triggering response.

Syslog Severity Actions

Syslog is the universal log protocol (RFC 5424). Priority is calculated as Facility × Severity. Most production environments categorize the 8 severity levels into three distinct action zones.

Alert (0-3) Emergency, Alert, Critical, Error. Requires immediate human action.
Log (4-6) Warning, Notice, Informational. Saved for searching and auditing.
Drop (7) Debug. Developer only, usually dropped to save storage space.

Lab Environments & Protocol

To build a portfolio, analysts must deploy configurations across various hypervisors and cloud platforms. Lab 3 requires documenting the deployment with precise screenshots and adhering strictly to legal and ethical frameworks.

💻 VirtualBox

Local desktop hypervisor ideal for endpoint isolation.

+ Free, offline, instant snapshots.
- Hard to emulate a complex router fabric.

🖧️ GNS3

Network emulator integrating real device images.

+ Real router/firewall execution.
- Extremely RAM-hungry for large topologies.

☁️ AWS Academy

Cloud environment for realistic production deployments.

+ Access to CloudWatch, GuardDuty.
- Time-limited credits, dependent on internet.

⚠️ Legal & Ethical Reminders

Cyber-defense tools become cyber-attack tools in unauthorized hands. The difference is consent in writing.

  • Authorized scope only: Never scan or monitor the campus network or your home LAN without explicit owner permission.
  • Legislation: Unauthorized access violates Canadian Criminal Code s.342.1, US CFAA, and EU 2013/40/EU.
  • Safety: Always paste suspicious URLs into VirusTotal before downloading.
