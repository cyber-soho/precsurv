Install gns3/endhost
in GNS3

A lightweight Alpine-based Docker container with nmap and essential networking tools — no heavy QEMU VM required.

Alpine Linux base
~10 MB image
Requires GNS3 VM
nmap preinstalled
Installation steps
1
Make sure GNS3 VM is running

Docker containers require the GNS3 VM (VMware or VirtualBox). Start it before doing anything else — the local server alone won't work.

⚠ If you only have a local GNS3 install without the VM, Docker containers are not supported. Download the GNS3 VM from github.com/GNS3/gns3-vm.
2
Open Docker container preferences

In the GNS3 GUI, navigate to:

path
Edit → Preferences → Docker Containers → New
3
Enter the image name

Select New image, then type the Docker Hub image name exactly as shown below, and click Next.

docker image 
gns3/endhost
4
Configure the container

Fill in the wizard fields:

settings
Name          endhost   (or any name you prefer)
Adapters      1
Start command /bin/sh
Console type  telnet

Click Next then Finish.

5
Drag into topology — image pulls automatically

Find endhost in the End Devices toolbar. Drag it into your topology — GNS3 will trigger a docker pull automatically (internet required, happens once only).

ⓘ You can also manually pull it in the GNS3 VM CLI with: docker pull gns3/endhost
6
Start, console in, and run nmap

Right-click the node → Start → Console. Assign an IP address and start scanning.

// quick start commands

shell 
# assign an IP to the interface
ip addr add 192.168.1.10/24 dev eth0
ip link set eth0 up

# ping sweep the subnet
nmap -sn 192.168.1.0/24

# scan open ports on a specific host
nmap -sV 192.168.1.1

# check SSH key exchange algorithms
nmap --script ssh2-enum-algos 192.168.1.1

Why Docker over Kali CLI
gns3/endhost (Docker)
  • ✓ ~10 MB image size
  • ✓ Boots in <1 second
  • ✓ Shared kernel — no overhead
  • ✓ Run many instances at once
  • ✓ Nmap + 12 other tools
Kali Linux CLI (QEMU)
  • ~ Several GB disk image
  • ~ Slower boot, more RAM
  • ~ Full OS emulation
  • ~ Requires KVM enabled
  • ✓ Full Metasploit + toolset
